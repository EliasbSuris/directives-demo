Directive composition example

Create a directive extending from one or more directives to reduce number or attributes on a component tag and reuse directive's code to expand functionalities.

This component use aorCompositeExample (CompositeExampleDirective) and aorCompositeExample this directive use:

  • ClickHandlerDirective: to notify click event through an event emitter
  • MouseoverHandlerDirective: to notify mouseenter & mouseleave events through an event emitter

Host directives also can receive input data, e.g: applying a click limit to the counter

Dynamic component example

Directives can be used in dynamically created components, thanks to hostDirective
